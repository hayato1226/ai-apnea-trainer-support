# Support Information / サポート情報

App Store Connect に登録するためのサポートURL（Support URL）として、以下のいずれかの方法を用意することをお勧めします。

## 方法 A: Google フォーム (推奨)
最も簡単で管理しやすい方法です。
1. Google フォームを作成し、「お問い合わせ内容」「返信先メールアドレス」の項目を作ります。
2. フォームの「送信」ボタンから共有リンクを取得します。
3. そのリンクを App Store Connect の **Support URL** に設定します。

## 方法 B: SNS アカウント (X / Twitter)
アプリ専用の公式アカウントなどがあれば、そのプロフィールページのURLを設定することも可能です。ただし、DMを開放しておく必要があります。

## 方法 C: GitHub Issues
開発者向けには便利ですが、一般ユーザーにはハードルが高いため、今回のアプリのターゲット層によっては不向きかもしれません。
